### Usage

These screenshots are ready for upload to App Store Connect:

1. Navigate to App Store Connect → Your App → App Store tab
2. Scroll to "App Preview and Screenshots"
3. Select "6.7" Display" section
4. Upload screenshots in order (01, 02, 03, 04)
5. First screenshot (01-url-qr-code.jpg) will appear in App Store search results

### App Store Screenshot Guidelines

According to Apple's guidelines:
- Screenshots must represent actual app content
- No placeholder or mockup content
- Must be from the current version being submitted
- Can add text overlays and frames (optional)
- First screenshot is most important (shows in search)
