## Common Issues & Solutions

### Issue: App name already taken

**Solution:**
- [ ] Try variations (Pro, Plus, Code, etc.)
- [ ] Use 321 branding (321 QR Generator)
- [ ] Add descriptive words (Quick, Fast, Simple)

### Issue: Screenshots not exact dimensions

**Solution:**
- [ ] Use simulator for exact sizes
- [ ] Check image properties before upload
- [ ] Export at 100% scale from design tool

### Issue: Privacy policy URL not working

**Solution:**
- [ ] Verify GitHub Pages is enabled
- [ ] Check URL is public (not requiring login)
- [ ] Test in incognito/private browser window
- [ ] Ensure HTTPS is working

### Issue: Build not appearing in App Store Connect

**Solution:**
- [ ] Wait 15-30 minutes for processing
- [ ] Check email for upload errors
- [ ] Verify bundle ID matches
- [ ] Complete export compliance

### Issue: Validation errors

**Solution:**
- [ ] Read error message carefully
- [ ] Check bundle ID consistency
- [ ] Verify app icon has no transparency
- [ ] Ensure Info.plist has required keys
