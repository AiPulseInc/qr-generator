## App Store Optimization Tips

### Title Strategy
- Include "QR Code" in the title if possible (high search volume)
- Keep it simple and descriptive
- Check competitor apps to find gaps

### Screenshot Strategy
- First screenshot is MOST important (appears in search results)
- Show the most impressive feature first
- Use captions/text overlays to explain features
- Include all 5 QR code types across screenshots
- Show both light and dark mode

### Description Strategy
- Use bullet points and emojis for scannability
- Lead with benefits, not features
- Include search keywords naturally (qr code, generator, wifi, etc.)
- Update promotional text seasonally or for campaigns

### Ratings & Reviews
- Encourage satisfied users to leave reviews
- Respond to all reviews (positive and negative)
- Address issues mentioned in negative reviews in updates

## Notes

1. **App Name Availability**: Check in App Store Connect before finalizing. If your preferred name is taken, try variations with symbols or additional descriptive words.
